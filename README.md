<br/>

**MediaSnap** is a *fast, clean, and privacy-respecting* video downloader for **TikTok**, **Instagram**, and **Facebook** — powered by `yt-dlp` and built on a **Node.js + Express** backend.

No re-encoding. No watermarks where possible. No login. ***Just paste and download.***

## How It Works

<br/>

<div align="center">

```
  PASTE LINK  ──────────►  FETCH INFO  ──────────►  DOWNLOAD
```

</div>

<br/>

**`1` &nbsp; Paste Link** &nbsp;&nbsp; — &nbsp;&nbsp; *Copy the video URL from TikTok, Instagram, or Facebook and paste it in the box.*

**`2` &nbsp; Fetch Info** &nbsp;&nbsp; — &nbsp;&nbsp; *MediaSnap instantly pulls the title, thumbnail, duration, and file size in real time.*

**`3` &nbsp; Download** &nbsp;&nbsp; — &nbsp;&nbsp; *Tap **Download MP4** or **Save to Device**. Best quality. Clean. Done.*

## Why MediaSnap

<br/>

> **Lightning Fast** &nbsp; — &nbsp; *Direct streaming from source. No re-encoding, no queues, no waiting.*

> **No Watermark** &nbsp; — &nbsp; *TikTok videos download **clean** — no platform branding, no overlay.*

> **No Login Required** &nbsp; — &nbsp; *Paste. Fetch. Download. No accounts, no sign-ups, no data collected.*

## Tech Stack

<br/>

<div align="center">

| Layer | Stack |
|:---|:---|
| **Backend** | Node.js &nbsp;+&nbsp; Express |
| **Download Engine** | yt-dlp *(always latest)* |
| **Output** | MP4 &nbsp;/&nbsp; Best Audio — *no re-encoding* |
| **Frontend** | iOS 26 Liquid Glass UI |
| **Hosting** | Netlify |
